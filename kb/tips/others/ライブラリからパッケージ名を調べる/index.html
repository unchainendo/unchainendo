ライブラリからパッケージ名を調べる
2020/02/01

「依存性の欠如」等のエラーでライブラリの不足を指摘された場合、不足しているライブラリから対応するパッケージ名を確認したくなる。 そんなときはyum whatprovidesで確認することができる。

yum whatprovides libpam.so*
読み込んだプラグイン:product-id, search-disabled-repos, subscription-manager
This system is not registered with an entitlement server. You can use subscription-manager to register.
pam-1.1.8-22.el7.i686 : An extensible library which provides authentication for applications
リポジトリー        : rhel77-eus
一致          :
Provides    : libpam.so.0
Provides    : libpam.so.0(LIBPAM_MODUTIL_1.0)
Provides    : libpam.so.0(LIBPAM_MODUTIL_1.1)
Provides    : libpam.so.0(LIBPAM_EXTENSION_1.1)
Provides    : libpam.so.0(LIBPAM_1.0)
Provides    : libpam.so.0(LIBPAM_MODUTIL_1.1.3)
Provides    : libpam.so.0(LIBPAM_EXTENSION_1.0)
Provides    : libpam.so.0(LIBPAM_EXTENSION_1.1.1)

参考: RPM依存ライブラリ不足の問題はYUMに聞こう！
